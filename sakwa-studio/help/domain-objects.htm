Domein object

Domein objecten gebruik je voor het structureren van je variabelen zie voorbeeld hieronder of om een sub model in je beslismodel te linken.
Voeg domein objecten toe via: model menu -> Add -> Domain object. Hierna kan in het eigenschappen venster de eigenschappen van het domein object veranderd worden.


Domein object eigenschappen




  • Het veld ShortName definieert een alias voor het domein object.
Merk op dat de subdomein objecten Naam en 'Post adres' weergegeven worden als 'Klant.Naam' en 'Klant.Post adres', de namen van alle domein objecten tot aan de variabele.

Voor extra leesbaarheid is het mogelijk om een domein object een verkorte naam te geven in de eigenschappen editor.
De subdomein naam wordt dan afgekort tot deze naam.



  • Het veld Model koppelt het domein object aan een ander beslismodel.

Sub model

Om je beslismodellen leesbaar te houden is het handig om sub modellen te gebruiken.
Je koppelt een beslismodel aan een domein object wat daarna dient als plaatshouder voor de variabelen in het sub beslismodel.



Als je op de ellips knop klikt in het Model veld dan opent de 'Link sub model' dialoog venster .



Je ziet alle variabelen die in het huidige beslismodel  en het gekozen beslismodel aanwezig zijn.

Alleen de variabelen die uniek zijn in het gekozen beslismodel worden gelinkt.

Na op de knop 'Link' te klikken worden de variabelen uit het gekozen beslismodel in het domein object gekopieerd zoals in de volgende afbeelding.
Variabelen die in beide beslismodellen gedefinieerd zijn worden niet gekopieerd.



Het icoon van het domein object verandert zodat je ziet dat deze de link naar een sub beslismodel bevat.
  • Het veld Methods definieert methoden die het domein object kan uitvoeren.
    Als je op de ellips knop klikt dan kun je de geselecteerde methoden aanpassen.


In het dialoog venster kun je de methoden van het domein object wijzigen.

    • De lijst Selected methods toont alle methoden van het domein object.
      Je kunt één of meerdere waarden uit deze lijst selecteren en verwijderen met de knop .

    • De lijst Available laat de methoden van alle domein objecten uit het beslismodel zien
      en gedefinieerde standaard methoden zie configuratie.
      Je kun één of meerdere elementen selecteren en toevoegen aan de lijst Selected methods, met de knop .

      Let op, je ziet alleen ongebruikte standaard methoden aangeduid worden met .

    • Via het veld New method kun je nieuwe waarden toevoegen aan de lijst Selected methods.
      Let op, laat het New method veld leeg als je een lege waarde wilt toevoegen.

      Let op, een nieuwe methode voeg je toe door op het  icoon te klikken of door de <Enter> toets te gebruiken.

    • Met en kun je de volgorde van de methoden veranderen.
