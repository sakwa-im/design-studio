Variable definition

Variable definitie

Voeg variabele definities toe via: model menu -> Add -> Variabele definition. Hierna kan in het eigenschappen venster de naam van de variabele definitie en het variabele type veranderd worden.

Mogelijke variabelen typen zijn:
  • character, dit is het standaard variabele type.
  • numeric
  • enumeration
  • date
  • boolean

Afhankelijk van het variabele type toont het eigenschappen venster extra eigenschappen om de variabele definitie verder te specificeren.

Veranderen van variabele eigenschappen

Hieronder is de variabele eigenschappen editor getoond.

s

  • Het Name veld spreekt voor zich.

  • In het veld Default value definieer je de waarde die gekozen wordt als je de variabele gebruikt in een assignment of branch element.
    Dit waarde die je hierin kunt invullen hangt af van het variabele type.

  • In het Variable Type veld kies je het type van de variabele definitie.
    De eigenschappen editor verandert in de editor voor het gekozen variabele type.

Character variable

Character variabelen kunnen letters, cijfers en leestekens bevatten.
Dit variabele type is de default en wordt automatisch gekozen wanneer een nieuwe variabele definitie toegevoegd wordt.

Numeric variable

Deze variabelen kunnen getallen bevatten van: -79228162514264337593543950335 tot 79228162514264337593543950335.
Het zijn 128 bit getallen (-7.9 x 1028 tot 7.9 x 1028) / (100 tot 28)


  • Het veld Maximum definieert de hoogst mogelijke waarde.

  • Het veld Minimum definieert  de laagst mogelijke waarde.

  • Het veld Sequence definieert de grote van de stappen voor de standaard waarde.
    Hieronder zie je het gebruik van Sequence.
    Let op, de sequence werkt alleen als de waarde 20% of minder van het bereik van de numerieke variabele is.

Enumeration variable

Enumeratie variabelen bevatten een variabele bereik bestaand uit een vaste set.



  • Het veld Elements bevat alle mogelijke waarden van de enumeratie variabele.
    Als je op de ellips knop klikt dan kun je het variabele bereik aanpassen.

  • Het veld Default value definieert een waarde uit het variabele bereik.



In het dialoog venster kun je de elementen in het bereik van de enumeratie variabele wijzigen.

  • De lijst Selected elements toont alle waarden in het bereik van de variabele.
    Je kunt één of meerdere waarden uit deze lijst selecteren en verwijderen met de knop verwijder.

  • De lijst Available laat de namen van alle branches en assignment elementen uit het beslismodel zien.
    Je kun één of meerdere elementen selecteren en toevoegen aan de lijst Selected elementen.

  • Via het veld New value kun je nieuwe waarden toevoegen aan de lijst Selected elementen.
    Let op, laat het New value veld leeg als je een lege waarde als element wilt toevoegen.

    Let op, een nieuw element voeg je toe door op het plus icoon te klikken of door de <Enter> toets te gebruiken.

  • Met de Up en Down knoppen kun je de volgorde van de waarden in de lijst Selected elementen veranderen.

Boolean variable

Het bereik van een booleaanse variabele is True (waar) en False (onwaar).
