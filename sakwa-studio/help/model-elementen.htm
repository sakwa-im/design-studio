Decision model

Het beslismodel is het basis element.

Variable definitions

Dit element is altijd aanwezig in het beslismodel. Hier worden variabele definities en domein objecten opgeslagen.

Variable definition

Variabele definities definiëren variabelen die in het beslismodel gebruikt kunnen worden.

Voeg variabele definities toe via: model menu > Add > Variabele definition. Hierna kan in het eigenschappen venster de naam van de variabele definitie en het variabele type veranderd worden.

Mogelijke variabelen typen zijn:
  • character, dit is het standaard variabele type.
  • numeric
  • enumeration
  • date
  • boolean

Afhankelijk van het variabelen type toont het eigenschappen venster extra eigenschappen om de variabele definitie verder te specificeren.

Klik hier voor uitgebreide uitleg over variabelen.

Domain object

Domein objecten stellen je in staat om structuur in de variabelen definities aan te brengen of om naar een sub model te linken.

Klik hier voor uitgebreide uitleg over domein objecten.

Assignment

Assignment nodes definiëren een waarde toekenning aan een variabele.

Voeg assignment elementen toe via: model menu > Add > Assignment. Hierna kun je in het eigenschappen venster een variabele selecteren en een waarde kiezen.

De beste volgorde is:

  1. Kies het domein object of een algemene variabele.
  2. Kies de variabele uit het domein object
  3. Kies een waarde uit het variabele domein

Bulk operations > Assignments

Er wordt een dialoog venster getoond waarin alle variabelen worden getoond. Hierna kunnen meerdere variabelen gekozen worden en de volgorde waarin ze aan het beslismodel toegevoegd worden bepaald.


Branch

Een branch node definieert het verloop door het beslismodel indien er aan de branch voorwaarde voldaan wordt.

Voeg een branch element toe via: model menu > Add > Branch. Hierna kun je in het eigenschappen venster een variabele selecteren en een waarde kiezen waarmee vergeleken wordt.

Bulk operations > Branches

Er wordt een dialoog venster getoond met alle enumeratie variabelen. Na selectie van een van deze variabelen kan een of domein elementen gekozen worden die als branch aan het model worden toegevoegd.


Gebruik de <Shift> en <Ctrl> toesten om de gewenste domein elementen te kiezen.

Bulk operations > Import

Met deze bulk operatie kun je variabele en domein objecten uit een ander beslismodel in je beslismodel importeren.



Deze knop opent een dialoog venster om een beslismodel te laden van disk.

Deze knop wist de waarde van het Source file invoer veld.

Je ziet alle variabelen die in het huidige beslismodel  en het gekozen beslismodel aanwezig zijn.
Als de variabele in beide beslismodellen gedefinieerd zijn dan geeft de kleur de mate van overeenkomst aan:

    Geen of verschillen die simpel op te lossen zijn.

    Grotere verschillen die je zelf moet oplossen.

    Blokkerende verschillen die niet oplosbaar zijn.

Je zult voor je kunt importeren eerst een import actie moeten kiezen uit het actie menu:



  • Use mine en Use theirs is te kiezen als de variabele in beide beslismodellen gedefinieerd is.

  • Skip is te kiezen als de variabele die alleen in het gekozen beslismodel gedefinieerd is.

Het icoon krijgt een indicatie van de gekozen import actie.
