Branch

Een branch element definieert het verloop door het beslismodel.
Dit pad wordt gevolgd als de waarde van een variabele voldoet aan de waarde die in de branch gedefinieerd is.

Voeg een branch node toe via: model menu -> Add -> Branch. Hierna kun je in het eigenschappen venster de naam aanpassen.

Branch eigenschappen



Variabele selection

  • In het veld Domain object kies je het domein object van de variabele die de branch gebruikt.
  • In het veld Variable kies je de variabele die de branch gebruikt.
  • Als je een domein object hebt gekozen dan kun je hier enkel variabelen uit dat domein object kiezen.

  • In het Value veld zet of kies je de waarde uit het bereik van de variabele, zie variabelen voor meer details.

    • Als een variabele gedefinieerd is dan kun je enkel waarden uit het bereik van deze variabele invullen.

    • Als enkel een domein object is gedefinieerd dan kun je methoden van het domein object invullen.

    • Als er geen domein object of variabele gedefinieerd is dan kun je hier elke waarde invullen.

Value selection

  • In het veld Value: domain object kies je het domein object van de value die de branch test.
  • In het veld Value: variable kies je de variabele die de branch test.
  • Als je een Value: domein object hebt gekozen dan kun je hier enkel variabelen uit dat domein object kiezen.

  • In het Value: value veld zet of kies je de waarde uit het bereik van de variabele, zie variabelen voor meer details.

    • Als een Value: variabele gedefinieerd is dan kun je enkel waarden uit het bereik van deze variabele invullen.

    • Als enkel een Value: domein object is gedefinieerd dan kun je methoden van het domein object invullen.

    • Als er geen domein object of variabele gedefinieerd is dan kun je hier elke waarde invullen.
