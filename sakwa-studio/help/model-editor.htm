Beslismodel editor

Snel naar: index, model elementen

Een beslismodel bestaat uit elementen die hiërarchisch gerangschikt zijn. De volgende afbeelding toont alle element typen en de onderlinge hiërarchie.



Element selectie

Je selecteert een element door er met de muis op te klikken. Het element wordt nu geselecteerd en de eigenschappen editor geeft je de mogelijkheid om element eigenschappen te bekijken en te veranderen.

Meerdere elementen selecteren

Je selecteert meerdere elementen wanneer je de <Ctrl> toets indrukt wanneer en op elementen in het beslismodel klikt.



De volgende regels gelden voor meerdere element selectie:
  1. Je kunt alleen meerdere elementen selecteren die direct onder hetzelfde element zitten.

  2. Als je de eerste maal klikt dan wordt het element ook geselecteerd.

  3. Als je opnieuw klikt dan wordt het element niet meer geselecteerd.

  4. Klik je zonder de <Ctrl> toets dan wordt een eventuele selectie opgeheven en selecteer je enkel het element.

Element Toevoegen

Wanneer je met de rechte muis knop op een element klikt dan opent het element menu.
Afhankelijk van het geselecteerde element kun je een keuze maken uit een of meerdere menu items.


De grijze menu items dienen als categorie label, klikken hierop heeft geen effect.

Kijk hier voor element details.

Element verwijderen

Kies uit het element submenu de optie in categorie 'Remove' of druk op <Delete>

Element

Wanneer je een model element geselecteerd hebt dan wordt het element en alle onderliggende elementen verwijderd.

Remove: decision model

Wanneer je het beslis model geselecteerd hebt dan word het hele model verwijderd.

Je wordt gewaarschuwd als je beslismodel veranderingen heeft ten opzichte van de opgeslagen versie, of wanneer je bij een nieuw beslismodel kiest voor verwijderen.


Elementen verplaatsen

Elementen kunnen in het beslismodel verplaatst worden door ze te verslepen naar een nieuwe plek.

Elementen kopieren

Elementen kunnen in het beslismodel gekopieerd worden door ze te verslepen naar een nieuwe plek met ingedrukte <Ctrl> toets.

Copy and paste van beslis elementen met toets combinaties komt in een volgende versie beschikbaar.

Expand from here

Vouwt alle elementen uit vanaf het geselecteerde element of druk op <Ctrl> + pijl naar rechts.

Collapse from here

Vouwt alle elementen dicht vanaf het geselecteerde element of druk op <Ctrl> + pijl naar links.

Reload decision model

Sluit het geselecteerde beslismodel en opent deze opnieuw.
Als het model gewijzigd is dan wordt dit met een dialoog venster gemeld zie Remove: decision model.
